# SmartWatch

>In this repository, a web-based SmartWatch had been designed using Bootstrap, Flex and various tools

### Features
                    
Note that, this web-base aplication have features such as
                    
-Message/Chat box                    
-Spotify Music player using iFrame                    
-Timer with laps
